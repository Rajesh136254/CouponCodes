# RedSorm Early Bird Coupon Page

## Setup

1. Save the RedSorm logo as `static/logo.png`
2. Install deps: `pip install flask`
3. Run: `python app.py`
4. Open: http://localhost:5000

## Admin

- URL: http://localhost:5000/admin
- Email: `admin@redsorm.in`
- Password: `RedSorm@Admin2024`

Change the password hash in `app.py` ‚Üí `ADMIN_PASS_HASH` before going live.
